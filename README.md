# 🖨️ Smart Printer Web

**A sleek, fully functional web application** that brings the powerful printing experience of Smart Printer directly to your browser.  
Print photos, PDFs, documents, and more — instantly, beautifully, and from any device.

## ✨ Features
- 📤 **Drag & Drop Upload** – Supports images, PDFs, Office docs & more  
- 🖼️ **Real-time Preview** – See your files exactly as they will print  
- 🔌 **Simulated Printers** – HP, Canon, Epson, Brother & more  
- 📡 **Wi-Fi & Bluetooth Toggle** – Fake connection status for realism  
- ⚙️ **Advanced Settings** – Paper size, color mode, copies, orientation, quality  
- 📚 **Beautiful Templates** – Birthday cards, business cards, calendars & more  
- 📱 **100% Mobile-Friendly** – Bootstrap 5 + responsive design  
- 🌐 **SEO Optimized** – Meta tags, keywords, fast loading  
- 🖨️ **One-Click Print** – Triggers realistic print simulation + success modal  

## 🛠️ Technologies Used
- 🔨 **HTML5** – Semantic structure  
- 🎨 **CSS3** – Modern gradients, animations & dark theme  
- 📱 **Bootstrap 5** – Fully responsive & mobile-first  
- ⚡ **JavaScript + jQuery** – Interactive file handling & printing logic  
- 🚀 **No external dependencies** beyond CDNs (ready to run)  

## 🚀 How to Run
1. Download all 4 files into the **same folder** (no subfolders)  
2. Open `index.html` in any browser  
3. Start uploading files and printing!
